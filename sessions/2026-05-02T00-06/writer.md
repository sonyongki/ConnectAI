## 📚 Deliverable 1: 수능 학습 계획 생성기 프롬프트 체인 명세서 (Prompt Chain Specification)

본 문서는 AI 엔진(LLM API 호출 시 사용)이 사용자 데이터를 받아 **JSON 포맷**의 맞춤형 학습 로드맵을 정확하게 출력하기 위한 최종 설계도입니다. 개발자에게 전달될 '명세서' 형식으로 작성되었습니다.

---

### 🎯 목표 (Objective)
사용자가 입력한 현재 성적, 목표 대학, 가용 시간을 기반으로 수능일까지 남은 기간 동안의 과목별 주차별/일별 학습 계획을 체계적으로 생성하고 JSON 형태로 반환하는 것이 목적입니다. 단순히 조언이 아닌, **'실행 가능한(Actionable)' 시간표**를 제공해야 합니다.

### ⚙️ 입력 스키마 (Input Schema)
AI 모델에 전달될 필수 변수 목록과 예시 값입니다.

| 변수명 | 타입 | 설명 | 필수 여부 | 예시 데이터 |
| :--- | :--- | :--- | :--- | :--- |
| `current_grade` | String | 현재 학년 (예: 2학년) | O | "고등학교 2학년" |
| `target_university` | String | 목표 대학 및 전형 (예: 연세대, 학생부종합/수능유형) | O | "연세대학교 / 수능+학종" |
| `mock_scores` | JSON Object | 최근 모의고사 성적 (과목별 백분위 또는 등급) | O | `{"국어": 320, "영어": 145, "수학": 85}` |
| `available_hours` | Integer | 일일 평균 가용 학습 시간 (시간 단위) | O | 6 |
| `d_day_days` | Integer | 수능까지 남은 총 일수 | O | 200 |

### 🧠 처리 지침 (Processing Instructions - AI Prompt Core Logic)
AI 모델에게는 다음의 사고 과정을 거치도록 명령해야 합니다.

1.  **진단 및 격차 분석:** `mock_scores`와 `target_university`에서 요구하는 평균 점수/등급을 비교하여, **가장 시급하게 보완해야 할 취약 과목(Weakest Links)**과 그 정도를 파악합니다.
2.  **가중치 부여 (Weighting):** 목표 대학의 전형별 중요도와 현재 성적 격차에 따라 각 과목/영역(`국어: 비문학, 수학: 미적분`)에 학습 가중치를 할당합니다. (예: 수능 유경험자에게는 고난도 문제 풀이에 높은 가중치 부여).
3.  **기간 분배:** `d_day_days`를 '기초 다지기(Foundation)' 기간, '심화 적용(Deep Dive)' 기간, '최종 점검(Final Review)' 기간으로 3단계로 나눕니다.
4.  **계획 수립:** 할당된 가중치와 시간을 고려하여 주차별/일별 학습 목표를 구체적인 행동 지침("OOO 유형 기출 5회 분석", "수학 개념서 전 범위 복습")으로 작성합니다.

### 💻 출력 스키마 (Output Schema - JSON Format)
반드시 아래의 엄격한 JSON 구조를 준수하여 결과를 반환해야 합니다. 이 포맷은 프론트엔드에서 Gantt Chart 또는 Stepper 컴포넌트로 직접 파싱되어야 합니다.

    }
  },
  "weekly_plan": [

      "week_number": 1,
      "target_period": "YYYY.MM.DD ~ YYYY.MM.DD",

        {"day": "월요일", "subject": "국어", "task": "비문학 기출 분석 3세트 (독해 구조화)"},
        {"day": "화요일", "subject": "수학", "task": "미적분 개념서 전 범위 복습 및 오답노트 작성"},
        // ... 나머지 요일 추가
      ]
    },
    // 최대 15주차까지 반복하여 배열 구성
  ]
}

## ✨ Deliverable 2: MrBeast 스타일 유튜브 스크립트 초안 (스크립트 완성)

**영상 주제:** "수능 만점생들이 절대 말해주지 않는, 1등급을 만드는 비밀 공부 루틴"
**스타일:** 극단적 후킹, 고조된 서사(Storytelling), 시각 자료 강조.

### [Title Draft] (클릭률 최대화 버전)
😱 **[경고] 이 영상 보면 수능 만점됩니다. 1등급은 '이것'부터 시작해야 함 (만점생의 비밀 루틴)**

### [Video Script]

**▶️ [0:00 - 0:10] 후킹 구간 (THE HOOK) - 시각적 충격과 질문 던지기 (가장 중요!)**
*(화면: 스톱모션 효과로 성적표 점수들이 급격하게 올라가는 모습. 긴박한 비트의 음악)*

**나레이터 (극적이고 자신감 넘치는 톤):** "당신이 지금 공부하는 방식, **틀렸을 확률 99%**입니다."
*(화면: 'XX점'에서 갑자기 빨간색 X 표시가 뜨는 성적표 클로즈업)*
**나레이터:** "만점을 받는 사람들은 절대 알려주지 않아요. 그들이 아는 단 하나의 비밀 루틴이 있습니다. 바로, **[핵심 키워드 삽입: 예) '문제 풀기 전 뇌 활성화', '시간 배분 공식']**입니다."
*(화면: 타이틀 카드가 크게 등장하며 궁금증 극대화)*

**▶️ [0:10 - 0:30] 서사 도입 (SETUP & PROBLEM) - 공감과 절망감 조성**
*(장면 전환: 일반적인 학원 강의실, 지쳐 보이는 학생들. BGM은 다소 우울하고 진지하게.)*

**나레이터:** "모두가 '열심히' 하라고 합니다. 하지만 열심히만 한다고 만점되는 게 아닙니다. 수능 공부는 **'효율성 전쟁'**입니다. 시간 대비 성과를 극대화하는 방법을 모르면, 당신의 노력은 그냥 소음일 뿐이에요."
*(자료 화면: 수많은 기출문제들이 흩날리는 모습)*
**나레이터:** "저도 그랬습니다. 무조건 오래 앉아있는 게 답인 줄 알았죠. 그런데 만점생들은 완전히 다른 전략을 쓰고 있었습니다."

**▶️ [0:30 - 2:00] 본론 전개 (THE SOLUTION) - 단계별 공개 및 증명**
*(BGM이 다시 고조되고, '비밀'을 푸는 듯한 기대감을 주는 사운드 효과 추가.)*

**(Section 1: 국어/영어 – 정보 처리 방식의 혁신)**
**나레이터:** "첫 번째 비밀은 **읽지 않는 겁니다.** 모든 지문을 읽으려고 하면 시간만 버려요. 핵심 정보를 빠르게 '구조화'해서 메모하는 스킬이 필요합니다."
*(시연 화면: 실제 국어 비문학 독해 팁 시각 자료와 함께 보여주기)*

**(Section 2: 수학/과학 – 개념의 연결고리 찾기)**
**나레이터:** "두 번째 비밀은 **'왜?'라는 질문입니다.** 공식을 암기하지 마세요. 그 공식이 어떤 원리로 작동하는지, 근본적인 메커니즘을 이해해야 합니다."
*(시연 화면: 수학 문제 풀이 과정을 역추적하며 핵심 원리를 도식화)*

**(Section 3: 전반적인 루틴 – 시간 관리의 극대화)**
**나레이터:** "그리고 가장 중요한 세 번째 비밀. 바로 **'진단-계획-점검' 사이클을 의무화하는 것.** 단순히 문제를 푸는 게 아니라, 내가 어디가 약하고, 다음 주에 뭘 해야 하는지 데이터로 확인해야 합니다."

**▶️ [2:00 - 2:30] 클라이맥스 및 CTA (CALL TO ACTION) - 제품 전환 유도**
*(화면: 강렬한 배경음악과 함께, 지금까지 말한 모든 '비밀'을 한 번에 담아내는 듯한 UI가 등장)*

**나레이터:** "이 세 가지 전략만 알아도 당신의 점수는 확실히 올라갑니다. 하지만... 이 비밀들을 **당신의 현재 성적(진단)에 맞춰서 100일 로드맵으로 짜주는 건, 아무나 할 수 없습니다.**"
*(화면: '맞춤형 AI 로드맵'이라는 문구가 크게 강조됨)*
**나레이터:** "이 영상에서 제가 보여드린 모든 원리, 이 전략들을 당신의 현재 성적과 목표 대학에 맞춰서 **실시간으로 분석하고, 주차별 플랜을 짜주는 시스템.** 바로 SKYIN 웹사이트에 있습니다."

**(마지막 멘트 - 결연한 의지):** "더 이상 막막하게 공부하지 마세요. 지금 바로 설명란의 링크를 클릭해서 당신의 로드맵을 확인하세요. **당신의 점수가 달라집니다.**"
