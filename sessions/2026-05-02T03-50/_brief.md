# 📋 작업 브리프

**원 명령:** 1시간내로 웹페이지 초안 구축해서 local에서 review 및 ui design 결정합시다. 그리고 유투브 시나리오 작성해서 제출해주세요.

## 요약
시간 제한이 있는 만큼, 웹페이지의 기능적 초안 구축과 유튜브 콘텐츠 제작을 동시에 진행합니다. 웹페이지는 'Gap 시뮬레이션' 기능을 핵심으로 하는 MVP 구조를 목표로 합니다.

## 분배
- **💻 Developer**: 웹사이트 페이지의 기본 HTML/CSS 뼈대(Wireframe)를 로컬 환경에서 즉시 구축하십시오. 필수 기능은 '성적 및 목표 입력 → Gap 진단 결과 출력'에 초점을 맞추고, 결제 게이트웨이 연동을 위한 임시 더미 API 호출 구조까지 포함하여 최소한의 작동 가능한 초안을 1시간 내로 완성해야 합니다.
- **🎨 Designer**: 개발자가 만든 웹페이지의 뼈대(Wireframe)를 기반으로, '학부모 공포감 자극'과 '신뢰성 있는 해결책 제시'가 결합된 UI 디자인 초안을 제작하십시오. 특히 Gap 진단 결과 화면에 시각적 충격과 동시에 따뜻한 희망을 주는 컬러 팔레트와 핵심 메시지 배치를 결정해야 합니다.
- **✍️ Writer**: 유튜브 영상의 스크립트 초안을 작성하십시오. 주제는 '2028 대입 변화에 따른 학부모/학생들의 예상치 못한 Gap'을 제시하는 방식으로, MrBeast 스타일의 극적인 오프닝(Hook)과 문제 제기 → 해결책 제시 (SKYIN 서비스 연결) 흐름이 포함되어야 합니다. 시간은 7~10분 분량으로 구성합니다.
- **🔍 Researcher**: 웹페이지와 유튜브 콘텐츠에 활용할 최신 입시 트렌드 키워드를 최종 검토하고, 특히 'Gap 진단'의 근거가 될 수 있는 2028 대입 제도 변경 관련 데이터 및 학부모들이 가장 불안해하는 핵심 문장(Pain Point Copy) 5가지를 정리하여 개발자/작가에게 즉시 전달하십시오.
