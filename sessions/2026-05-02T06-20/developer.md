### 🛠️ 1. 데이터 파이프라인 개요 및 난이도 분석

**핵심 기능:** 사용자 성적/목표대학 $\rightarrow$ AI 기반 Gap 진단 (문제 인식) $\rightarrow$ 목표 대학 제안 + 맞춤형 주간 학습 로드맵 생성 (해결책 제시)

| 항목 | 평가 | 근거 및 기술적 이슈 |
| :--- | :--- | :--- |
| **기술 난이도** | ⭐️⭐️⭐️⭐️☆ (상) | 단순 데이터베이스 검색(Lookup)을 넘어, *성능 예측* 및 *논리적 추론(Inference)*이 필요함. (AI/ML 모델 경량화 또는 복잡한 규칙 기반 시스템 설계 필수) |
| **예상 소요 기간** | 3~4주 | 현재 백엔드 스택(FastAPI + Supabase)을 활용하여 '규칙 기반' MVP를 구현하는 기준으로 산정했습니다. (진짜 AI 모델 학습 시간 제외) |
| **핵심 위험 요소** | 데이터의 정제 및 구조화 | 수집된 교육 데이터를 단순한 점수가 아닌, **상대적 포지션(Percentile)**과 **성장 가능 영역**으로 재분류하는 로직 설계가 가장 중요하며 시간이 많이 소요됩니다. |

> 💡 **전략적 제언:** 초기 MVP 단계에서는 복잡한 ML 모델 대신, 사전에 구축된 방대한 데이터셋에 기반하여 '규칙(Rule-based)'을 정의하고 이를 API로 구현하는 것이 시간 대비 효율성이 높습니다. (ex: 만약 A과목 점수가 90점 이상이고 B과목이 70~80점 사이라면, 목표는 OO 전공으로 설정하고 부족한 영역은 C 과목 학습 집중).

### ⚙️ 2. 필수 API 구조 정의 및 스키마 명세

핵심 로드맵 생성 기능을 수행할 엔드포인트와 데이터 교환 규약을 다음과 같이 정의합니다. 이 사양에 맞춰 백엔드 개발을 진행할 예정입니다.

#### **Endpoint: `/api/v1/diagnose_roadmap`**
*   **메소드:** `POST`
*   **기능:** 사용자 입력 데이터(성적, 목표 등)를 받아 종합 진단 및 로드맵 생성.
*   **인증 요구:** (선택적) 세션 기반 인증 또는 추후 결제 시스템 연동을 위한 고유 ID 필요.

#### **A. Request Body Schema (입력값 정의)**

사용자가 웹페이지 폼에 입력하는 데이터를 구조화한 JSON 스키마입니다.

```json
{
  "student_info": {
    "grade": "고등학교 2학년", // 예: High School Year 2
    "major_interest": ["컴퓨터공학", "생명과학"], // 관심 분야 키워드 배열
    "current_gpa": 3.8, // 현재 내신 GPA (4.5 만점 기준)
    "subject_scores": { // 과목별 점수 및 등급 정보
      "국어": {"score": 92, "grade": "A"},
      "수학": {"score": 85, "grade": "B+"},
      "영어": {"score": 95, "grade": "A"},
      "탐구_과목1": {"score": 88, "grade": "B+"}, // 예: 물리학 I
      "탐구_과목2": {"score": 75, "grade": "C+"}  // 예: 화학 I
    }
  },
  "goal_input": {
    "target_university": "서울대학교", // 목표 대학 (Optional)
    "min_desired_gpa": 4.0     // 최소 희망 GPA (Optional)
  }
}
```

#### **B. Response Body Schema (출력값 정의)**

진단 결과와 학습 계획이 포함된 구조화된 JSON 스키마입니다. 이 데이터가 프론트엔드의 모든 UI 컴포넌트를 채웁니다.

```json
{
  "status": "success",
  "diagnosis_summary": { // 1. 종합 진단 및 Gap 분석 (사용자에게 가장 먼저 보여줄 핵심 메시지)
    "title": "종합 컨설팅 보고서: 데이터 기반 로드맵 제시",
    "core_gap_identified": "수학 과목의 심화 개념 이해 부족(Gap)", // 현재 문제점을 명확히 지적
    "strength_areas": ["영어 (어휘력)"], // 강점 영역 요약
    "suggested_direction": "인공지능 기반 공학 계열 진학 추천", // 방향성 제시
  },
  "roadmap": { // 2. 실행 가능한 로드맵 및 계획
    "recommended_universities": [
      {"name": "서울대학교", "fit_score": 85, "reason": "최적의 학업 역량과 목표가 부합함."},
      {"name": "카이스트(KAIST)", "fit_score": 78, "reason": "탐구 과목 심화 학습을 통해 도전 가능."}
    ],
    "learning_plan": { // 주간 계획 (Weekly Plan)
      "total_weeks": 12,
      "weekly_focus": [
        {"week": 1, "theme": "Gap 해소: 수학 개념 재정립", "subjects": ["수학"], "tasks": ["고난도 문제집 챕터 3까지 완료"]},
        {"week": 2, "theme": "내신 관리: 탐구 과목 심층 분석", "subjects": ["탐구_과목2"], "tasks": ["오답노트 작성 및 선생님 피드백 반영"]}
      ],
      "study_priority": [ // 우선순위 (High/Medium/Low)
        {"subject": "수학", "priority": "High"},
        {"subject": "탐구_과목2", "priority": "High"}
      ]
    }
  },
  "suggested_action": { // 3. 다음 단계 (가장 중요한 CTA: 유료 결제 유도)
    "message": "이 로드맵은 개괄적인 진단 결과입니다. 학부모님께서는 맞춤형 자료와 컨설턴트의 심층 코멘트를 받아보시려면 프리미엄 플랜을 확인하세요.",
    "cta_link": "/premium-report-purchase"
  }
}
```
