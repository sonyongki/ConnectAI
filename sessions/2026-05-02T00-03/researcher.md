## 🚀 II. 경쟁사 대비 SKYIN의 독점적 데이터 및 분석 관점 (USP 제안)

현재 시장의 대부분 컨설팅 서비스는 **'정적인 정보 제공(Static Data)'**에 머물러 있습니다. 저희 웹페이지가 단순한 정보를 나열하는 것을 넘어, **'지능형 예측 모델을 통한 동적 가이드라인'**을 제시해야 합니다.

### 1. 문제점 인식 (경쟁사 한계)
*   **대부분의 서비스:** "현재 성적 A $\rightarrow$ 목표 대학 B에 지원 가능합니다." (단순 매칭/IF-THEN 구조)
*   **결과:** 학생은 '가능'한 것만 보게 되고, 현실적인 학습 계획이나 개선 포인트를 놓치기 쉬움.

### 2. SKYIN의 독점적 USP 제안: **[다변수 시뮬레이션 기반 예측 로드맵]**
저희가 제공해야 할 것은 단순 점수가 아닌, **시간에 따른 성장의 '궤도(Trajectory)'**입니다. 웹페이지는 아래와 같은 고차원적인 분석 기능을 탑재해야 합니다.

| 구분 | SKYIN의 독점 기능 (USP) | 데이터/분석 근거 | 기대 효과 (사용자 경험) |
| :--- | :--- | :--- | :--- |
| **핵심 엔진** | **Multi-Factor Simulation Model (MFSM)**: 성적, 시간 투자량, 학습 방법론(A, B, C), 전형 유형을 변수로 넣어 합격 확률의 '범위'와 '필요 노력치'를 동적으로 시뮬레이션합니다. | 전문 컨설팅 회사가 보유한 **내부 데이터 기반 학과별/전형별 과거 합격생 데이터를 가중치화**하여 구축해야 합니다. (데이터 독점성) | "지금 이대로 가면 70% 확률로 이 대학에 갈 수 있으나, 주간 계획을 이렇게 조정하면 95%까지 끌어올릴 수 있습니다." $\rightarrow$ **실행 가능한 의지 부여.** |
| **컨텐츠 차별화** | **'Weak Point-Targeting Feedback Loop'**: 현재 성적에서 가장 점수 상승 효율이 높은 '취약 영역(Weak Point)'을 찾아내고, 그 취약점을 보완하는 학습 자료와 주간 미션을 자동으로 추천합니다. | 일반적인 과목 간의 상관관계를 넘어, **학습 방법론 자체를 분석**하는 메타-인지적 접근 방식 적용. | "수능 국어 점수를 올리려면 문법 공부가 아니라, '독서 지문 구조 파악' 훈련에 시간을 투자해야 합니다." $\rightarrow$ **전문성 극대화.** |
| **서비스 모델 연계** | **단계별 컨설팅 자료 제공 (Tiered Content)**: 웹페이지에서 얻는 기본 정보(4990원)를 넘어, MFSM이 제시한 '개인 맞춤형 주간 학습 계획표(9990원)'에 접근하도록 설계. | 명확하게 수익화 모델을 콘텐츠 깊이에 연결하고, 서비스 사용 자체를 컨설팅 경험으로 인식시킵니다. |
